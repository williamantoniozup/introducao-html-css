classes = flexibilidade, sempre que possível use CLASSES no seu css;

Não colocar texto maisculo direto no HTML, resolvemos isso com CSS

Display-inline não pega height e width
text-indent - joga o texto pra longe do elemento, e a imagem continua sendo
clicavel. Técnica image replacement.

pelo conteúdo. USe CLEAR pra empurrar pra baixo as demais divs. Clearfix, 
uso de clear para trabalhar com float.

Relative - relativo a posição original, mantem o fluxo da página.
Absolute - relativo a janela do meu navegador
